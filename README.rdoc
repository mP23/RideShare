== README

How to set up this application. 

    First clone the project from Github and install it on c9.
    Github link: https://github.com/slee264/RideShare
    Then do "bundle install --without production"
            "rake db:migrate"
    After that you can run "rails s -p $PORT -b $IP" to see the actual website. 
    
    Heroku link is here: https://salty-bastion-71342.herokuapp.com/
